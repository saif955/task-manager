Task manager App using the mern stack

Usage:

1. Edit the .env file to include your own URI.
2. Install the backend dependencies with:
   
   ```
   npm install.
   ```
   
4. Get into the frontend:
   
   ```
   cd frontend
   ```
   
6. Install the frontend:

   ```
   npm install
   ```
   
8. Go back to root:

   ```
   cd ..
   ```
   
10. to run the app:
    
    ```
    npm run server
    ```
